# Episodes

```text
episodes/player-slug/
  brief.json
  research.md
  script.csv
  prompts.csv
  images/       # generated locally, ignored
  audio/        # narration, ASR and timings, ignored
  renders/      # final videos, ignored
```

`script.csv` 是字幕和旁白文本真源。`body-timings.json` 只提供时间；ASR 文本不覆盖脚本。
